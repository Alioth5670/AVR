# AVR
Experiment of ATmega16

本项目为单片机课程关于ATmega的学习项目

This project is a learning project about ATmega for SCM course

Electronic alarm clock 使用的是ATmega16，压缩包里有LCD驱动可依据需求更改，其余文件使用的均为ATmega128


Electronic Alarm Clock USES ATmega16. The compression pack has an LCD driver that can be changed according to requirements. The rest files are used ATmega128
